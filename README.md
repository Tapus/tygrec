# :book: La Quête

Static website for Coline Tournus's "La Quête", a funny story featuring some of her friends.

## Link

See it [here](https://la-quete.thuault.com).

## Build

### Build image

```bash
./build/build.sh
```

### Deploy on Kubernetes

```bash
./build/deploy.sh
```

### Update deployment image on Kubernetes

```bash
./build/update.sh
```
